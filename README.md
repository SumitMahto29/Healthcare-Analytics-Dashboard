# Healthcare Analytics Dashboard

## Overview

This project is an interactive Healthcare Analytics Dashboard built using Power BI. The dashboard analyzes hospital data to provide insights into patient admissions, revenue generation, diagnosis trends, insurance coverage, patient satisfaction, and operational performance.

## Objectives

* Monitor key healthcare KPIs.
* Analyze patient demographics and diagnosis patterns.
* Track hospital revenue across different states.
* Evaluate admission and readmission trends.
* Understand insurance coverage distribution.
* Support data-driven decision-making in healthcare management.

## Dashboard Features

### Key Performance Indicators (KPIs)

* Total Patients: 50,000
* Total Revenue: ₹2 Billion
* Average Patient Satisfaction: 3.75/5
* Readmission Rate: 14.86%
* Average Length of Stay (LOS): 4.41 Days

### Visualizations

* Total Revenue by State
* Diagnosis Distribution by Department
* Diagnosis Distribution by Age Group
* Monthly Revenue Trend
* Insurance Coverage Analysis
* Admission Type Distribution
